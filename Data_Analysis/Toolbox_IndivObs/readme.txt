This folder contains the code from Asano et al. that was used in this study to generate the simulated observer population. 

To access the original code and its description, access: https://www.rit.edu/science/sites/rit.edu.science/files/2019-01/MCSL-Observer_Function_Database.pdf

To regenerate the simulated observers, run the function StartHere_MonteCarloObs.
